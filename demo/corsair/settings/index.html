// Corsair HQ — pipeline overview

Sources (read-only)
├─ Zoho Books      → invoices, bills, GL, VAT
├─ Bayzat          → headcount, salaries, attendance
├─ HubSpot         → contracts, accounts, sales pipeline
├─ Asana           → projects, milestones, status, owners
├─ Ticketstream    → festival ticket sales, escrow movements
└─ Bank feeds      → cash positions, transactions

ETL (hourly)
├─ Normalisation   → currency, VAT, period mapping
├─ Reconciliation  → invoice ↔ project ↔ cash
└─ Roll-up         → unit / month / year aggregates

Insight engine — Jeeves GPT
├─ Plan vs Actual deviation analysis
├─ DSO / cashflow / runway forecasting
├─ Project burn-rate & overrun detection
└─ Counterfactual narratives ("what if Aurora signs CO?")

Alert system
├─ Rule evaluation (per yml above)
├─ Severity escalation
└─ Multi-channel delivery (email / Slack / SMS / dashboard)

Output surfaces
├─ Corsair Dashboard (this UI)
├─ REST API for downstream consumers
└─ Weekly Markdown digest to founders
